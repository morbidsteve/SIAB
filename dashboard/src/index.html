Quick Start - Deploy Your App

Deploy your containerized application in seconds using the SIABApplication CRD:

apiVersion: siab.io/v1alpha1
kind: SIABApplication
metadata:
  name: my-app
spec:
  image: myregistry/myapp:v1.0
  replicas: 3
  port: 8080

  security:
    scanOnDeploy: true
    blockCriticalVulns: true

  auth:
    enabled: true
    requiredRoles: ["user"]

  ingress:
    enabled: true
    hostname: myapp.siab.local
    tls: true

Apply with: kubectl apply -f my-app.yaml
